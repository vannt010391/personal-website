## Further Customization

If you want to adjust the palette:

### Option 1: Lighter Version
- Primary: #FF9999, Secondary: #FFCCCC (more pastel)

### Option 2: Darker Version
- Primary: #E85D5D, Secondary: #F39999 (more saturated)

### Option 3: Alternative Warm Colors
- Primary: #FF6B6B, Secondary: #FFB3A1 (orange-red shift)

To change colors globally, update the hex values in:
- `static/css/blog-zen.css` (search for #FD7979 and #FDACAC)
- Template files (inline styles with gradients)
